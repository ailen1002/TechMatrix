###  Installation 

```
dotnet new install Avalonia.Templates
```

### Prerequisites 

- .NET 8.0
- Semi.Avalonia framework

### Steps 1. Clone the repository:   

```
git clone https://github.com/ailen1002/TechMatrix.git
cd TechMatrix
```

### Steps 2. Build and run the project:   

```
dotnet build
dotnet run
```
